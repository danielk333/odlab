PYOD¶

Release

0.1.0

Date

May 18, 2020

PYOD (pronounced “pi-o-d”) stands for PY Orbit Determination and is an open-source software for scientific research concerning orbit determination using observation data.

Simple example¶

import pyod

Install¶

pip install pyod

Testing¶

You need

  • pytest >= 5.3.5

pytest

If you want to test orekit propagator, that requires data to function, call pytest with the additional parameter orekit_data. Otherwise these tests are skipped.

pytest --orekit_data='/orekit-data-master.zip'
